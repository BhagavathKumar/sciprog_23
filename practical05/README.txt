This code is for Week 5 Practical.

------------
fibonacci_p5.c
------------

Calculates the Fibonacci series of upto the value n (provided by the user).

To complie:

gcc -o fib fibonacci_p5.c


--------------
archtanh_p5.c
--------------

Calculates the hyperbolic arc tangents using two methods:

1. Maclaurin Series
2. Natural Logarithms

To compile:

gcc -o arc archtanh_p5.c
